OMdb API

The Open Movie Database

The OMDb API is a RESTful web service to obtain movie information, all content and images on the site are contributed and maintained by our users.

Usage

Send all data request to:

http://www.omdbapi.com/?apikey=[YOURAPI]&

Poster API requests:

http://img.omdbapi.com/?apikey=[YOURAPI]&

Parameter

By ID or Title

Parameter Discription
i A valid imdbId(eg.tt1285016).
t Movie title to search for.
type Type of result to return.
y Year of release.
plot Return short or full plot.
r The data type to return.
callback JSONP callback name.
v API version.

By Search

Parameter Discription
s Movie title to search for.
type Type of result to return.
y Year of release.
r The data type to return.
page Page number to return.
callback JSONP callback name.
v API version.
